Coolmark Intranet

This page contains the case study of Coolmark Intranet which includes the Project Overview and Tools Used.

Project Overview

Coolmark is now one of the Netherlands' leading wholesalers in the field of refrigeration technology, climate technology, heat pumps and more.

I created their multi-page intranet using my Frontend Web Development skills to give them an intranet that looks good, is easy to navigate and has all the important data for this company.

Unfortunately, an intranet is not publicly accessible and, as you will understand, I cannot show the project.
