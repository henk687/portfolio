Project Overview

I decided to recreate the frontend of GitHub's official web app because I enjoyed recreating it and it was a great opportunity for me to demonstrate my frontend skills by rebuilding the UI and working with the GitHub API ended up being a great experience for me.

I used my frontend tools to build the UI that works on all types of devices, along with the GitHub API to display all GitHub profiles.
